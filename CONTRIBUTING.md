# Contributing

Improvements to accessibility, metadata quality, exports, documentation, testing, and researcher usability are welcome.

## Suggested workflow

1. Fork the repository.
2. Create a focused branch.
3. Test LitScope in a modern browser.
4. Keep personal notes, downloaded articles, and private research data out of commits.
5. Open a pull request describing the problem and the proposed change.

Please preserve the lightweight, local-first, no-account-required design.
