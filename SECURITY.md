# Security and privacy

## Reporting a problem

Use GitHub Issues for non-sensitive bugs. Do not post credentials, private notes, institutional access details, or sensitive research data publicly.

## Data handling

LitScope stores saved papers and notes in browser local storage. Do not enter confidential, personal, clinical, or unpublished sensitive information on a shared or unmanaged computer.

## External services

Search requests are sent directly to OpenAlex or Semantic Scholar. Their availability, privacy policies, metadata quality, and rate limits are outside this project's control.

## Secrets

LitScope does not require API keys. Never commit credentials, tokens, institutional login details, copyrighted article PDFs, or proprietary datasets to this repository.
